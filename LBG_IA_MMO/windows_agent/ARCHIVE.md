# Windows agent — ARCHIVE

Le dossier `windows_agent/` ne contient actuellement qu’un seul module actif :

- `windows_agent/Agent_IA/` : worker Windows HTTP (cible `C:\Agent_IA`) utilisé par `desktop_control`

S’il y a de nouveaux workers Windows à l’avenir, placer ici uniquement ce qui est **utilisé**.
Tout prototype/ancienne version devra être rangé dans un sous-dossier `ARCHIVE/` (ou documenté ici)
pour éviter la confusion.
